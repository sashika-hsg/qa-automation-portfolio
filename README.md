# QA Automation Portfolio

>Production-grade QA automation framework built with Playwright, Typescript,
>BDD AI integratio, and full QA lifecycle documentation.

## Status

Under active develoment - Phase 0 : Foundation

## Author

Sashika Samaragunaratne - Senior QA Analyst | Melbourne, Australia

## What this project demonstrates

- UI automation with Playwright and Page Object Model
- API automationwith schema validation
- BDD with Gherkin and Cucumber
- Database validation with SQLite
- AI-powered test data generation and failure analysis
- Full QA lifecycle from requirements to test closure
- CI/CD with Github actions and SAWS Cloudwatch monitoring
